# SyntaxWear - E-commerce de Tênis e Sneakers

## Descrição

SyntaxWear é um site de e-commerce estático dedicado à venda de tênis e sneakers de alta qualidade. O projeto apresenta um design moderno e responsivo, focado em oferecer uma experiência de compra intuitiva e atrativa para os amantes de calçados urbanos.

## Funcionalidades

- **Página Inicial Responsiva**: Design adaptável para dispositivos móveis e desktop.
- **Seções de Destaque**:
  - Hero section com chamada para ação.
  - Categorias de produtos (Casual, Esporte, Moderno, Futurista).
  - Grade de produtos em destaque.
- **Navegação Intuitiva**: Menu de navegação com categorias principais e acesso rápido.
- **Rodapé Completo**: Newsletter, links de navegação e ícones de redes sociais.
- **SEO Otimizado**: Meta tags e estrutura semântica para melhor indexação.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da página.
- **CSS3**: Estilização modular com variáveis CSS.
- **SVG**: Ícones e logos vetoriais para escalabilidade.

## Estrutura do Projeto

```
ecommerce-syntaxwear/
├── index.html                 # Página principal do site
├── README.md                  # Este arquivo

    ├── icons/                 # Ícones SVG (menu, redes sociais, etc.)

```

## Instalação e Uso

Como este é um site estático, não há necessidade de instalação de dependências ou servidor backend. Para visualizar o site:

1. Clone o repositório:
   ```
   git clone https://github.com/seu-usuario/ecommerce-syntaxwear.git
   ```

2. Navegue até o diretório do projeto:
   ```
   cd ecommerce-syntaxwear
   ```

3. Abra o arquivo `index.html` em seu navegador web preferido.

## Desenvolvimento

Para contribuir ou modificar o projeto:

- Edite os arquivos HTML em `index.html`.
- Modifique os estilos CSS nos arquivos correspondentes em `css/`.
- Adicione ou substitua imagens na pasta `images/`.

Certifique-se de testar as alterações em diferentes navegadores e dispositivos para garantir a responsividade.
